# 🚗 Hero Motors Budget Performance Dashboard

## 📌 Project Overview

This project is a Power BI Budget Performance Dashboard created for analyzing Hero Motors budget performance and trends. The dashboard helps track budget allocation, compare previous and current budgets, and analyze performance across states, months, and product groups.

---

## 🎯 Objectives

- Analyze Total Budget performance
- Monitor Year-To-Date (YTD) Budget
- Track Quarter-To-Date (QTD) Budget
- Track Month-To-Date (MTD) Budget
- Compare previous month vs current budget
- Analyze state-wise performance
- Understand product group contribution

---

## 📊 Dashboard Features

### KPI Cards
- Total Budget
- QTD Budget
- YTD Budget
- Average Budget
- MTD Budget

### Interactive Filters
- State Filter
- Year Filter
- Month Filter

### Visualizations Used
- KPI Cards
- Line Charts
- Bar Charts
- Trend Analysis
- Table Visualization

---

## 🛠️ Tools Used

- Power BI
- Power Query
- DAX
- Data Cleaning
- Data Modeling

---

## 📈 Key Insights

- Total Budget: ₹7.06M
- YTD Budget: ₹3.71M
- MTD Budget: ₹292.67K
- Highest budget contribution from "Others" product category
- Haryana had the highest budget allocation
